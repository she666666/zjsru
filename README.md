# 智能工厂数字孪生交互系统

> Digital Twin Factory Interactive System | 浙江树人学院期中作业

## 项目简介

本项目是一个基于 Three.js 开发的智能工厂数字孪生交互系统，在浏览器中实现了一个完整的三维工厂场景。系统支持鼠标浏览、对象点击信息展示、视角切换、图层控制等功能。

## 功能特性

### 🏭 场景对象

- **建筑物**: 主厂房、仓储中心、办公研发楼
- **道路**: 主干道、横向道路、人行通道（含道路标线）
- **设施设备**: 智能生产线、储罐组、冷却塔、起重机
- **植被**: 绿化树木、灌木丛

### 👁️ 视角控制

- **透视视角**: 默认三维视角，全方位观察工厂
- **鸟瞰视角**: 俯视整体布局
- **正交视角**: 正面观察建筑

### 🖱️ 交互功能

- **鼠标控制**: 拖拽旋转、滚轮缩放、右键平移
- **点击信息**: 点击建筑或设施显示详细信息面板
- **图层切换**: 可控制建筑物、道路、设施、植被、灯光的显示/隐藏
- **动画控制**: 可开启/关闭设施动画效果

### ⚡ 核心功能

1. **图文介绍**: 点击场景对象展示详细参数信息
2. **图层切换**: 灵活控制不同类型对象的可见性
3. **视角变换**: 多种视角无缝切换动画
4. **实时性能监控**: 显示 FPS 和对象数量

## 技术栈

- **Three.js**: WebGL 三维渲染引擎
- **OrbitControls**: 相机轨道控制
- **ES6 Modules**: 模块化 JavaScript
- **CSS3**: 现代UI样式

## 快速开始

### 方法一：直接打开

双击 `index.html` 文件在浏览器中打开（部分浏览器可能需要本地服务器支持）

### 方法二：本地服务器

```bash
# 使用 Python
cd factory-digital-twin
python -m http.server 8080

# 使用 Node.js
npx http-server -p 8080
```

然后访问: http://localhost:8080

## 使用说明

1. **浏览场景**: 鼠标左键拖拽旋转视角，滚轮缩放，右键平移
2. **查看信息**: 点击建筑或设施查看详细图文介绍
3. **切换视角**: 点击右侧控制按钮切换不同视角
4. **控制图层**: 点击底部复选框控制各类型对象的显示
5. **动画效果**: 点击"启用动画"按钮开启/关闭设施动画
